### Installation

#### The Easiest Way


#### Deploy to Heroku

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://www.heroku.com/deploy?template=https://github.com/NaveenTn49/uploader0404)

#### Run in your VPS
```sh
git clone https://github.com/prgofficial/URLuploader-With-Hotstar
cd URLuploader-With-Hotstar
pip3 install -r requirements.txt
# <Create config.py appropriately>
python3 bot.py
```

### Commands

* /start             - Check if bot is alive
* /help              - To know how the bot works
* /upgrade           - Nothing much here
* /deletethumbnail   - Clear saved thumbnail
* /showthumb         - Show saved thumbnail
